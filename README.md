# Taller 1 — Algoritmos Básicos en Python

**Autor:** Juan Diego Cartagena

Colección de 26 ejercicios introductorios en Python organizados en un menú interactivo. Cada algoritmo resuelve un problema concreto de lógica, matemáticas o administración básica.

---

## Requisitos

- Python 3.x instalado
- No se necesitan librerías externas (solo `math`, que viene incluida)

---

## Cómo ejecutar

1. Asegúrate de que **todos los archivos** estén en la **misma carpeta**:

```
📁 taller1/
 ├── index.py
 ├── 1.py
 ├── 2.py
 ├── ...
 ├── 25.py
 └── funciones.py
```

2. Abre una terminal en esa carpeta y ejecuta:

```bash
python index.py
```

3. Selecciona el número del algoritmo que quieres probar y sigue las instrucciones en pantalla.
